Commands

The full list of Syndicate Corporation's commands and beta commands will be listed here with detailed information on each command.



Syndicate Corporation Commands:


    sn!help: List of Syndicate's Commands.
    sn!info: Learn More About Syndicate.
    sn!invite: Invite Syndicate To Your Discord Server.
    sn!server: Information About Your Discord Server.


Syndicate Corporation's Utility Commands:


    Custom Prefix: If you don't like Syndicate's Default prefix (sn!).
    You can change it to anything you want by using sn!setprefix 'your_chosen_prefix'.
    To view your prefix use sn! guildprefix. The prefix by default is sn!.


Toggle-able Functions:




    XP-System: Everytime you type in a guild Syndicate will randomly generate XP and you will level up in the guild.
